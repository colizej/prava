# 🔍 PermisDeConduire-Online.be - Анализ безопасности

Скрипты для анализа структуры и тестирования защиты контента.

**Цель:** Понять, какой контент защищен, а какой доступен публично.

---

## 📂 Структура

```
permisdeconduire-online.be/
├── scripts/
│   ├── analyze_structure.py    # Анализ структуры сайта
│   ├── test_security.py         # Тест безопасности
│   └── scraper.py               # Скрапер (если контент публичный)
├── output/                      # Результаты
└── README.md                    # Этот файл
```

---

## 🚀 Использование

### Шаг 1: Анализ структуры сайта
```bash
cd scripts
python3 analyze_structure.py
```

Этот скрипт покажет:
- Структуру навигации
- Ссылки на теорию и вопросы
- Признаки защиты контента

### Шаг 2: Тест безопасности
```bash
python3 test_security.py
```

Проверяет:
- ✅ Доступен ли бесплатный контент
- 🔒 Защищен ли платный контент
- ⚖️ Правильно ли настроена защита

---

## 🔐 Этика и безопасность

### ✅ Что мы делаем:
- Анализируем ПУБЛИЧНЫЕ страницы
- Проверяем защиту платного контента
- Помогаем понять структуру сайта

### ✅ Вердикт после теста:
- Если контент **защищен** → скачивание возможно ТОЛЬКО с подпиской
- Если контент **публичный** → можно скачать (это публичная информация)
