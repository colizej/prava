# 🚗 Scraper Уроков ПДД - ReadyToRoad.be

Web scraper для извлечения теоретических материалов по ПДД с сайта readytoroad.be в структурированный JSON формат.

**⚠️ Важно:** Этот скрипт предназначен исключительно для личного образовательного использования с разрешения владельца ресурса.

---

## 📋 Что скрипт делает

- ✅ Извлекает **все 13 категорий** теоретических уроков (A-M)
- ✅ Сохраняет **структурированный контент** (заголовки, тексты, списки)
- ✅ **Скачивает изображения** локально
- ✅ Создает **иерархический JSON** с категориями и подкатегориями
- ✅ **Симулирует человека**: случайные задержки, реалистичные headers
- ✅ **Обработка ошибок** и логирование прогресса

## 🚀 Использование

### Базовый запуск
```bash
python3 scraper.py
```

Скрипт автоматически:
1. Создаст папку `output/`
2. Создаст подпапку `output/images/` для изображений
3. Начнет скрапинг всех 13 категорий
4. Сохранит промежуточные результаты в `lessons_data_partial.json`
5. Сохранит финальный результат в `lessons_data_complete.json`

### Ожидаемое время выполнения
- **~20-40 минут** (зависит от скорости интернета)
- Скрипт делает задержки 2-5 секунд между запросами
- Прогресс отображается в консоли в реальном времени

## 🔧 Решение проблем

### Ошибка: ModuleNotFoundError
```bash
pip install -r requirements.txt --upgrade
```

### Ошибка: Connection timeout
- Проверьте интернет-соединение
- Увеличьте timeout в коде: `timeout=60`

### Слишком медленная работа
- Уменьшите задержки (НЕ рекомендуется):
  ```python
  scraper.human_delay(min_seconds=0.5, max_seconds=1)
  ```

### Недостаточно места на диске
- Изображения могут занять ~100-300 MB
- Освободите место или отключите скачивание изображений

## 📝 Примечания

- ⏱️ **Задержки обязательны** для уважения к серверу
- 🔒 **Только бесплатный контент** - скрипт не обходит авторизацию
- 📸 **Изображения**: скачиваются только реальные изображения (не SVG placeholder'ы)
- 💾 **Промежуточное сохранение**: данные сохраняются после каждой категории
- 📊 **Логирование**: весь прогресс отображается в консоли
