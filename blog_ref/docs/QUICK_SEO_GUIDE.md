📊 **Причина:** Редизайн + Core Web Vitals на мобильных + Google переоценка

## 🚨 СРОЧНЫЕ ДЕЙСТВИЯ (эта неделя)

### 1. Исправить мобильную версию (КРИТИЧНО!)
```bash
# Проверить скорость загрузки:
open https://pagespeed.web.dev/
# Ввести: https://piecedetheatre.be

# Что проверить:
# - Largest Contentful Paint (LCP) < 2.5s
# - Cumulative Layout Shift (CLS) < 0.1
# - First Input Delay (FID) < 100ms
```

### 2. Найти 404 ошибки
```
1. Открыть: https://search.google.com/search-console
2. Индексирование → Страницы
3. Прокрутить до "Не найдено (404)" (29 страниц)
4. Экспортировать список
5. Добавить 301 редиректы в core/urls.py
```

### 3. Переотправить sitemap
```
1. Открыть: https://search.google.com/search-console
2. Файлы Sitemap → sitemap.xml
3. Нажать на три точки → "Запросить индексирование"
```
